## Recommended code workflow
[*raw_data_extraction_paramecium.R*](raw_data_extraction_paramecium.R), [*raw_data_extraction_didinium.R*](raw_data_extraction_didinium.R), [*raw_data_extraction_homalozoon.R*](raw_data_extraction_homalozoon.R) - processing scripts of raw COMTRACK data provided in the [COMTRACK data folder](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Data/comtrack). The output of each script is also provided in the [Data folder](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Data/).

[*noar_models.R*](noar_models.R) - fits Bayesian hierarchical models to *Paramecium caudatum* speeds and morphologies. The saved models can be found in the [Results folder](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Results/).

[*Figure2.R*](Figure2.R), [*Figure3.R*](Figure3.R), [*FiguresS1_4.R*](FiguresS1_4.R), [*TablesS1_2.R*](TablesS1_2.R)- generate the main figures available [here](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Results/figures), the [supplementary figures](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Results/figures/supplementary_figures), and supplementary tables [here](https://github.com/franzmatches/Full-length-and-high-res-predator-exposure/tree/main/Results/figures/supplementary_figures).
